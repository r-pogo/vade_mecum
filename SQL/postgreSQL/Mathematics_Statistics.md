# Basics of mathematics and statistics
| Operator | Opis |
|----------|------|
| + | Dodawanie|
| - | Odejmowanie |
| * | Mnożenie |
| / | Dzielenie (bez reszty) |
| % | Modulo (zwraca tylko reszte) |
| ^ | Potęgowanie |
| &#124;/ | Pierwiastek kwadratowy |
| &#124;&#124;/ | Pierwiastek sześcienny |
| ! | Silnia |
___
## Dodawanie, odejmowanie i mnożenie
```sql
SELECT 2 + 2;
SELECT 9 - 1;
SELECT 3 * 4;
```
